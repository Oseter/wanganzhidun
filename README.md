> 反骚扰 / 反恶意举报的**个人防御型取证与举报辅助**工具集。

## ⚠️ 使用原则（红线）

本体系所有组件严格遵循：

1. **仅用于反制恶俗攻击**，不对非恶意目标使用；
2. **不伪造证据**，所有举报基于真实记录；
3. **不改包、不爆破、不逆向工程**；
4. **不爬取非授权数据**；
5. 高级组件的反伤功能**必须经用户手动确认**才发起，工具不自动举报；
6. 用户端零代码可用；开发端代码公开、可审计。

任何滥用（诬告、诬陷、滥用举报机制）均违背本体系初衷，使用者自负其责。

## 当前开发重点：网安智盾（存护命途）

跨平台防御组件：

- **Windows**（`The-Preservation/Cyber-Shield/Windows`）：PC 后台监测 + 自带录屏取证，PyInstaller 打包为 `WangAnZhiDun.exe`。
- **Chrome 扩展**（`Chrome-Extension`）：一键填充 12377 / 腾讯卫士 等举报表单（规划）。
- **Android**（`Android`）：通知监听 + 无障碍录屏取证（规划）。

详见各子目录 README。
