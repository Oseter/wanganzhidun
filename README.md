### Windows 版（当前重点）

常驻系统托盘，监听 QQ/微信 通知栏，关键词触发 **截图 + 自带循环缓冲录屏 + 证据归档（AES 加密）+ 标准弹药举报草稿**。

- 录屏**无需安装 OBS**，纯 CPU 软编码，零外部应用依赖；
- 图形界面（tkinter，统一 Tk 根，跨线程安全）：
  - **主仪表盘**：盾牌品牌头、监听/录屏/加密状态、本会话取证/证据/反伤统计、最近事件列表、快捷操作；
  - **配置窗**：六页签（监测/证据/录屏/举报邮件/反伤/标准弹药），保存即热更新，无需重启；
  - **系统托盘**：真盾牌图标，「显示主窗口 / 证据目录 / 设置 / 暂停继续 / 退出」；
  - **反伤确认弹窗**：展示攻击摘要与可编辑条款，带倒计时，必须手动确认才发起反伤；
- 详见 `The-Preservation/Cyber-Shield/Windows/README.md`。

一键打包：`Windows/build.bat` → 产出 `dist\WangAnZhiDun.exe`。
