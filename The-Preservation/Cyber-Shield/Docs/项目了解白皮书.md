## 三、网安智盾是什么

**定位**：个人防御型取证 + 官方渠道举报辅助工具。  
**场景**：恶俗方打你号、点你号、炸你群 → 网安智盾自动取证 → 你确认 → 走 12377/腾讯卫士反向举报。

**核心闭环**：
```
监听通知栏 → 关键词命中 → 截图+录屏 → 证据归档（AES加密）→ 弹窗确认 → 反伤举报
```

### 三端交付

| 平台 | 技术栈 | 状态 | 交付物 |
|------|--------|------|--------|
| Windows | Python 3.11 + tkinter + PyInstaller + SQLite + OpenCV | ✅ 可构建 | `WangAnZhiDun-Setup.exe` |
| Android | Kotlin + Room + MediaProjection + KSP | ✅ 可构建 | `WangAnZhiDun.apk` |
| Chrome | JavaScript MV3 | 🚧 规划 | `wang-an-zhi-dun.crx` |

### 标准弹药格式（全体系共享）

```
目标账号 | 时间 | 违规内容 | 对应条款 | 证据附件
```

这是整个体系的"数据协议"，所有命途组件都用这个格式传递举报信息。

## 五、工作流与工具链

### GitHub 仓库

- **地址**：`https://github.com/Oseter/wanganzhidun`
- **分支策略**：单 `main` 分支，直接 push
- **Token**：需要 classic GitHub token（有 repo 全权限），token 值单独提供给 AI

### CI/CD

- **Windows**：push 到 main → GitHub Actions `windows-latest` → Python 3.11 → PyInstaller onedir → Inno Setup 6 → Release（`WangAnZhiDun-Setup.exe` + `WangAnZhiDun-portable.zip`）
- **Android**：push 到 Android 目录变更 或 手动 `workflow_dispatch` → `ubuntu-latest` → JDK 17 + Gradle 8.9 → `assembleDebug` → Release（`WangAnZhiDun.apk`）

### Release 发布

- **测试版**：`https://github.com/Oseter/wanganzhidun/releases/tag/v1.1`（prerelease）
- 三个文件：Setup.exe（54MB）、portable.zip（80MB）、apk（6MB）

## 八、常见坑位（踩过的雷）

| 问题 | 原因 | 解决 |
|------|------|------|
| GitHub push 403 | OAuth token scope 为空 | 改用 classic token |
| CI 无法访问 github.com | 沙盒 DNS 劫持到 198.18.0.12 | 开公网（198.18.0.21） |
| Inno Setup ISCC 失败 | choco 版不含中文语言包 | 删除 `installer.iss` 中 `[Languages]` 段 |
| ISCC 找不到 exe 目录 | PyInstaller 产出是目录不是文件 | Source 改为 `dist\WangAnZhiDun\*` |
| APK 编译失败 | `android.media.MediaProjection` 路径错误 | 改为 `android.media.projection.MediaProjection` |
| APK 编译失败 | `Json.encodeToString(List<String>)` 类型不匹配 | 显式 `ListSerializer(serializer<String>())` |
| APK 编译失败 | `ArrayDeque.takeLast()` receiver mismatch | 先 `toList()` 再 `takeLast` |
