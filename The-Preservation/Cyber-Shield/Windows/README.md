## 功能

| 模块 | 说明 |
|------|------|
| 通知监听 | 通过 Windows 通知中心监听 QQ/微信 弹窗（winrt UserNotificationListener） |
| 关键词引擎 | 命中配置关键词即触发取证 |
| 截图 | 命中后多显示器截图 |
| 自带录屏 | **无需安装 OBS**，后台线程循环缓冲，命中时落盘最近 N 秒 MP4 |
| 证据归档 | 按标准弹药格式（目标账号/时间/违规内容/对应条款/证据附件）打包 |
| 本地加密 | 证据 AES 加密，防窃取篡改 |
| SQLite | 本地事件日志 |
| 反伤确认 | 检测到攻击弹出确认框，你确认后才生成举报草稿/发送 |
| 系统托盘 | 常驻后台，右键菜单控制 |

## 运行（开发模式）

```bat
pip install -r requirements.txt
python main.py
```

## 打包为 exe（零代码用户最终交付）

双击 `build.bat`，自动在 `dist\WangAnZhiDun.exe` 产出单文件程序。
首次运行会在系统设置中授权"读取通知"。

> 录屏为纯 CPU 软编码，**不依赖显卡**（无 NVENC 也可运行），适合老机型。

---

## 配置

所有行为改 `config.ini` 即可，无需碰代码：

- `monitor.keywords`：触发关键词（逗号分隔）
- `recorder`：自带录屏帧率 / 缓冲时长 / 分辨率缩放 / 显示器
- `evidence`：证据保存路径、是否加密
- `anti_strike`：是否启用反伤、确认超时
- `email`：举报邮件通道（可选，发往官方举报邮箱）
